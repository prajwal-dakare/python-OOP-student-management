# Student Management System

## Overview

This project is a simple Student Management System developed using Python. It demonstrates Object-Oriented Programming (OOP) concepts and the use of Python data structures.

## Features

* Classes and Objects
* Inheritance
* Lists
* Dictionaries
* Sets
* Tuples
* File Handling
* Exception Handling

## Technologies Used

* Python 3

## Project Structure

* `student_management.py` - Main Python program
* `students.txt` - Stores student records
* `README.md` - Project documentation

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

```bash
python student_management.py
```

## Learning Outcomes

This project helped in understanding:

* Object-Oriented Programming (OOP)
* Inheritance in Python
* Data Structures
* File Operations
* Error Handling
